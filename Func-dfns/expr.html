Functions Expressions

A JavaScript function can also be defined using an expression.
A function expression can be stored in a variable

After a function expression has been stored in a variable, the variable can be used as a function.

JavaScript function can defined with a built in JavaScript function constructor. Function()

You actually dont't have to use a function constructor.

Function expression can be made "self-invoking"

function expressions will excecute automatically if the expression is followed by ().
You have to add parathesis around the function to indicate that it is a functio expression

Functions can be treated as values.

x = myFunction(7,5) or x = 35

In both cases x becomes a number with a value of 35.
