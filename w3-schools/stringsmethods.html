Extracting String parts

There are 3 methods for extracting a part of a String

  1. Slice(start, end)
  2. substring(start, end)
  3. Substr(start, length)

JS strings slice()

It extracts parts of a string and returns the extracted part ina new string end not included

If the parameter is negative, the position is counted from the end of the string.

If you ommit the second parameter, the method will slice out the rest of the string.

Or counting from the end.

JavaScript string substring()

substring() and Slice() are similar.
The difference is that in substring() values less than 0 are treated as 0.
