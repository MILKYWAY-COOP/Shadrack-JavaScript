The <hr> tag is used to seperate contents in an HTML page.

The poem problem

HTML ignore spaces added to any paragraph. And the poem will display on a single line.
Solution is the <pre> element.
It defines performatted text, the text inside it is displayed in a fixed-width font and it preserves both spaces and line breaks.
