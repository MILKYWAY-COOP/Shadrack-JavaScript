Waiting for a Timeout

JavaScript setTimeout() method

JavaScript setTimeout() property.
It calls a function after anumber of miliseconds.
The setTimeout is only excecuted once.

Wait for 5 seconds (5000 miliseconds) for this page to display.
It will display within the <hr> element.

If you need repeated excecutions use setInterval() instead.
Use the clearTimeout() method to prevent the function from starting.
To clear a timeout use the id returned from setTimeout().

myTimeout = setTimeout(function, miliseconds);

Then you can stop the excecutions by calling clearTimeout():

clearTimeout(myTimeout);
