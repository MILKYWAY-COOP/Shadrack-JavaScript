Functions in Javacript

A javascript function is a block of code designed to perform a particular task.
A javascript function is excecuted when something invokes (call) it.

Javascript Function Synthax

It is defined with the function keyword, followed by a name, followed by a parantheses.
Functions can contain letters, diguts, underscores, and dollar signs (same rules as variables)
Function parameters- are listed inside the parantheses () in function definition.
Function arguments- are the values received by the function when it is invoked.

Function invocation

The code inside the function will excecute when "something" invokes it.

  • When an event occurs (eg a user clicks a button).
  • When it is invoked from javascript code.
  • Automatically(self invoked).
